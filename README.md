# 🛡️ Network Intrusion Detection System

## 📌 Project Overview

The **Network Intrusion Detection System (NIDS)** is a Machine Learning-based project developed to detect malicious network activities and classify network traffic as **normal** or **intrusion**. The project performs data preprocessing, feature engineering, model training, and evaluation to improve cybersecurity by identifying potential attacks.

---

## 🎯 Objectives

* Detect malicious network traffic.
* Improve network security using Machine Learning.
* Classify normal and attack traffic with high accuracy.
* Evaluate different ML algorithms for intrusion detection.

---

## 🚀 Features

* Data preprocessing and cleaning
* Feature selection and transformation
* Exploratory Data Analysis (EDA)
* Machine Learning model training
* Intrusion detection and classification
* Model performance evaluation
* Visualization of results

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/techportfolio3737-eng/Network-Intrusion-Detection-System.git
```

2. Open the project folder

```bash
cd Network-Intrusion-Detection-System
```

3. Install required libraries

```bash
pip install pandas numpy matplotlib scikit-learn jupyter
```

4. Open the notebook

```bash
jupyter notebook
```

---

## ▶️ How to Run

1. Launch Jupyter Notebook.
2. Open `Network Intrusion Detection System.ipynb`.
3. Run all notebook cells in sequence.
4. Review the model evaluation results and visualizations.
